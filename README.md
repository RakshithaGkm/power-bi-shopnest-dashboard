# ShopNest Business Analytics Dashboard

## 📊 Overview
This project analyzes e-commerce sales data to identify trends and business insights.

## 🛠 Tools Used
- Power BI
- DAX
- Power Query

## 🔍 Key Insights
- Revenue trends across regions
- Customer purchasing behavior
- Delivery performance analysis

## 📌 Outcome
Built an interactive dashboard to support data-driven decision making.
